Help System

The Help System plug-in, org.eclipse.help, defines two extension points ("contributions" and "contexts") which allow individual plug-ins to contribute online help and context-sensitive help for their components. A typical plug-in would make use of these extension points.

For more advanced applications, there is also an extension point  ("support") for plugging in a a help system UI.

Extension Points

The following extension points can be used to extend the help system:
  • org.eclipse.help.contributions
  • org.eclipse.help.contexts
  • org.eclipse.help.support
