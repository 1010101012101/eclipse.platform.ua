Welcome to Eclipse help

Click on one of sections in the Contents frame on the left to browse that section's contents. Each section loads a different navigation tree.

To return to this view, click the Table of Contents button, above the Contents frame.

Searching

To search the entire documentation set, enter a query in the Search field. To narrow down which sections are searched, click Advanced.

Synchronizing

After you run a search and find a topic you were looking for, click the Synchronize button to match the navigation tree up with the current topic. You might also find it useful to synchronize after following in-topic links.

Navigating by keyboard

You can navigate the help system by keyboard using the following key combinations:

  • To move to the next topic in the left frame, press Tab.
  • To move to the previous topic , press Shift+Tab.
  • To scroll the left frame up and down press the Up and Down arrows. To scroll all the
  • way up or down press Home or End.
  • To go back press Alt+Left arrow; to go forward press Alt+Right arrow.
  • To go to the next frame press Ctrl+Tab. There are quite a number of frames in the help system.
  • To move to previous frame, press Shift+Ctrl+Tab.
  • To print the current page or active frame, press Ctrl+P.
