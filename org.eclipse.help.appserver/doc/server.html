server

Identifier

org.eclipse.help.appserver.server

Description

Internal extension point to plug in an application server for help and other eclipse plugins.

Markup

   <!ELEMENT server EMPTY>
   <!ATTLIST server
     class   CDATA #REQUIRED
     default (true | false) "false""false"
   >

  • class - The contributors to this extension point must implement the org.eclipse.help.appserver.internal.IWebappServer interface.
  • default - When set to true, this is the default application server. Normally, only the platform contributed default implementation should set this attribute to true.

Example


<extension id="appserver" name="Tomcat" point="org.eclipse.help.appserver.server">
      <server class="org.eclipse.tomcat.internal.TomcatAppServer" default="true" />
</extension>

API Information

The supplied class must implement the org.eclipse.help.internal.appserver.IWebappServer interface. Methods in that interface control starting and stopping of the application server, as well as starting and stopping of a web application.

Supplied Implementation

The Eclipse Platform provides an implementation of this extension point using the Apache Tomcat application server.
